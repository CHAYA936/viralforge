**ViralForge**

AI-powered content creation platform for social media creators. Generate captions, scripts, hashtags, and viral content ideas — completely free.

**Features**

- AI Caption Generator - Platform-specific captions with customizable tone
- Reels/Shorts Script Generator - Complete video scripts with timing
- Hashtag Booster - Trending and relevant hashtags based on niche
- Viral Trend Finder - Current trending topics and content ideas
- Content Calendar - Visual planning and scheduling interface
- Thumbnail Text Maker - Compelling text overlays for thumbnails
- Viral Hook Generator - Attention-grabbing opening lines

**Tech Stack**

- React 18 + TypeScript
- Vite - Fast build tool with PWA support
- Tailwind CSS - Utility-first styling
- Framer Motion - Smooth animations
- Shadcn/ui - Accessible component library

**Getting Started**

1. Install dependencies: npm install
2. Start development server: npm run dev
3. Build for production: npm run build

**Target Audience**

Instagram creators, YouTubers, TikTok/Shorts/Reels creators, meme pages, personal brands, and small business pages.
